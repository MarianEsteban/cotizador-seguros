# ME Seguros - Cotizador rápido

## ¿Qué es este proyecto?
Es una web simple para captar consultas de personas interesadas en cotizar distintos tipos de seguros de ME Seguros.

## Objetivo
Permitir que cualquier persona complete un formulario corto, cargue datos específicos según el seguro que necesita y envíe la consulta directamente por WhatsApp.

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript (vanilla)
- Compatible con GitHub Pages

## ¿Cómo publicarlo en GitHub Pages?
1. Subir estos archivos al repositorio de GitHub.
2. Ir a **Settings > Pages**.
3. En **Build and deployment**, elegir:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (o la rama principal) y carpeta `/root`
4. Guardar cambios.
5. GitHub generará una URL pública (puede tardar unos minutos).
